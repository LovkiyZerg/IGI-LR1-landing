# Сайт-визитка Максима Юна

Сайт фехтовальщика ИСБ (Исторический Средневековый Бой) из Москвы.

## О проекте
Персональный сайт-визитка для бойца ИСБ и автора YouTube-канала.

## Технологии
- HTML5
- CSS3
- Font Awesome иконки

## Структура
- `index.html` - основная структура сайта
- `style.css` - все стили и адаптация под мобильные
